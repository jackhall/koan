# Simplify dispatch

Collapses the duplicated taxonomies, parallel channels, and consumer-side
re-matching inside the dispatch subsystem
([`src/machine/execute/dispatch.rs`](../../src/machine/execute/dispatch.rs) +
[`dispatch/`](../../src/machine/execute/dispatch/)) so each classification and
resolution concept has exactly one owner. The payoff is extensibility with a
compiler backstop: adding an eager part shape, a binder form, or a resolution
state becomes a one-site change the type system polices, instead of a
convention held in sync across four files.
